# Portrait images

Add your three portrait photos here with these exact filenames:

- `portrait-1.jpg` — back layer (largest, most rotated)
- `portrait-2.jpg` — middle layer
- `portrait-3.jpg` — front layer (hero focal point)

Supported formats: `.jpg`, `.jpeg`, `.png`, or `.webp` (update paths in `index.html` if you use a different extension).

Recommended size: at least **800×1000px** (portrait orientation) for sharp display on retina screens.
