# Portrait images

Gallery order is **left → right** in `index.html` (Fiat, trench coat, brownstone). Use these exact filenames:

| File | Scene | Visual cues |
|------|--------|-------------|
| `portrait-fiat.jpg` | Fiat street | Light-blue knit outfit; mint-green Fiat 500; brownstone block behind |
| `portrait-trench.jpg` | Trench coat | Tan double-breasted trench; patterned headscarf; cat-eye sunglasses |
| `portrait-brownstone.jpg` | Brownstone steps | Light-blue knit outfit; red door; stone steps |

**Do not** rely on numbered names (`portrait-1.jpg`, etc.) or Desktop exports named `portrait-1` / `portrait-2` — those labels do not match this mapping.

Supported formats: `.jpg`, `.jpeg`, `.png`, or `.webp` (update paths in `index.html` if you change extension).

Recommended size: at least **800×1000px** (portrait orientation) for sharp display on retina screens.
